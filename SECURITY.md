## Security Policy

#### Supported Versions

Only the latest version of ConfigurationAPI is actively supported.
Older versions may not receive security updates.

#### Reporting a Vulnerability
If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately:
- Contact: spexx@spexx.dev

---

#### What to Report

Please report:
- Remote code execution risks
- File system vulnerabilities
- Unsafe config handling
- Data leaks or exposure
- Thread-safety issues leading to corruption

---

#### Response Time
- Initial response: within a few days
- Fix timeline: depends on severity

---

#### Responsible Disclosure
Please allow time for a fix before publicly disclosing vulnerabilities.

Thank you for helping keep the project secure.
